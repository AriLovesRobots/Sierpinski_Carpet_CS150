# Sierpinski_Carpet_CS150
Simple GUI application of a Sierpinski Carpet generator based on the Stroustrup C++ GUI library.

You simply enter a number that represents the depth of the Sierpinski Carpet and then press the New Depth button to generate your Sierpinski Carpet of desire!

You can build the file with CMake (CMake file included).

You will also need the FLTK dependencies to be able to run this.
